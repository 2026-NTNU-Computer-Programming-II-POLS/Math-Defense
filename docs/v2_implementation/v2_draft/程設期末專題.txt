2. Towers
* Magic : {(1, 2, 3 次項的方程式), (Trigonometric Functions), (Logarithmic Function)} n=1
* Radar : 
   * Type-A : 範圍內掃描式傷害 (no target)
   * Type-B : 攻速高、傷害低 (can select target)
   * Type-C : 攻速低、傷害高 (can select target)
* Matrix : 
   * 兩塔為有效的一組；各塔預設一範圍 MR；各塔 (x,y) 將轉為矩陣元素 [x, y]
   * 兩塔範圍重疊部分受到矩陣相乘之結果傷害 (can select target)
   * 攻擊將以雷射鎖定同一目標，傷害將隨時間遞增
* Limit : 塔放置的x值 = a，(選擇/填充題) / (x - a)，依結果有不同傷害
   * 對敵人造成傷害: D(+∞) > D(+C)
   * 治療敵人(哈哈笨蛋): H(-∞) > H(-C)
   * Calculus : 給3~5個函式，選擇一次微分/二次微分/積分，依結果生成不同寵物
   * 若為 0 ，則該塔被移除 (無補償金；活該)
   * 若為常數，則失效
   *  : 緩速
   *  : 攻速高
   *  : 傷害高
   * 每波次可花錢選擇是否變更當前函數 (對其進行微分或積分)

         4. Location of the Start and End points

         * B-path: Trigonometric Functions
         * C-path: Logarithmic Function
         * path-type1: {A-path} n = 2-5
         * path-type2: {A-path, B-path} n = 2-3

         * Legal placement location: 隨機生成，不覆蓋 path
